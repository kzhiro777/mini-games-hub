# Mini Games Hub

ブラウザで遊べる3つのミニゲームを集めたハブアプリケーションです。

## 🎮 ゲーム一覧

### 1. テトリス
クラシックなテトリスゲーム
- **操作**: 
  - ← → : ブロック移動
  - ↑ : 回転
  - ↓ : 高速下降
- **機能**: スコア、レベル、ライン数の追跡
- **目標**: できるだけ多くのラインを消す

### 2. ブロック崩し
ボールでブロックを破壊するゲーム
- **操作**: マウス移動でパドルを操作
- **機能**: 
  - 複数の難易度レベル（色別）
  - スコア追跡
  - 3回失敗でゲームオーバー
- **目標**: すべてのブロックを破壊

### 3. アングリーバード風ゲーム
スキルベースのターン制シューティングゲーム
- **操作**: スキルボタンをクリック
- **スキル**: 
  1. 🚶 Walk : 長く歩く
  2. 💥 Spread Shot : 3方向に発射
  3. 💣 Explosion : 着弾地点で爆発
  4. 🔫 Pierce : 貫通力の強い弾
  5. 🔥 Flame : 着弾地点に炎効果
- **クリア条件**: 敵の80%以上を撃破して次レベルへ進む
- **ステージ**: 5ステージ

## 🛠️ 技術スタック

- **Phaser 3**: ゲーム開発フレームワーク
- **HTML5 + CSS + JavaScript**: フロントエンド
- **LocalStorage**: スコア・ゲーム状態の保存

## 📂 ファイル構成

```
mini-games-hub/
├── index.html                 # メインHTMLファイル
├── styles.css                # スタイル定義
├── js/
│   ├── main.js               # ゲーム初期化
│   ├── config.js             # Phaser設定
│   ├── utils/
│   │   └── StorageManager.js # セーブ/ロード機能
│   └── scenes/
│       ├── MenuScene.js      # メイン画面
│       ├── TetrisScene.js    # テトリス
│       ├── BlockBreakerScene.js # ブロック崩し
│       └── AngryBirdScene.js # アングリーバード風
```

## 🚀 使い方

1. リポジトリをクローン
```bash
git clone https://github.com/kzhiro777/mini-games-hub.git
cd mini-games-hub
```

2. ブラウザで `index.html` を開く
```bash
# または簡易サーバーで実行
python -m http.server 8000
# http://localhost:8000 にアクセス
```

3. ゲーム選択画面からプレイしたいゲームを選択

## 💾 セーブ機能

各ゲームのハイスコアとゲーム状態は LocalStorage に自動保存されます。
- テトリス: `tetris_scores`
- ブロック崩し: `blockBreaker_scores`
- アングリーバード: `angryBird_scores`

## 📊 スコアシステム

### テトリス
- ラインクリア: 100点 × レベル × クリア行数
- レベルは10ラインごとに上昇

### ブロック崩し
- ブロック破壊: 10点

### アングリーバード
- 敵撃破: 10点 × レベル
